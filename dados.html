Credenciais Omie
⚠ CORS: chamadas diretas ao Omie a partir do browser podem ser bloqueadas por CORS. Se o teste falhar com "CORS error", use a Supabase Edge Function abaixo como proxy intermediário. O upload Excel continua funcionando independentemente.
Supabase Edge Function (proxy CORS)
Cole este código em Supabase → Edge Functions → New function "omie-proxy"
// supabase/functions/omie-proxy/index.ts
import { serve } from "https://deno.land/std@0.168.0/http/server.ts";

serve(async (req) => {
  const body = await req.json();
  const res = await fetch("https://app.omie.com.br/api/v1/" + body.endpoint, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify(body.payload),
  });
  const data = await res.json();
  return new Response(JSON.stringify(data), {
    headers: {
      "Content-Type": "application/json",
      "Access-Control-Allow-Origin": "*",
    },
  });
});
